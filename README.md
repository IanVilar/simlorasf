# simlorafs
LoRa simulator to study Spreading Factor (SF) orthogonality.

## Prerequisite
Tool is implemented and tested for Python 3. 

Required Modules:
* matplotlib
* numpy
* scipy
* scikit-learn

## Usage
python3 main.py
